# Rstar

Raising Star (Rsrat) is a lite web-based B2C microservices seed project. It provides foundational back-office functionalities including login, page management, and Role-Based Access Control (RBAC). The project is built with Spring Cloud and Vue3.

Rstar is designed to be ready to use out of the box, enabling development teams to quickly bootstrap their projects. Its key features include:

* Easily customizable back-office UI themes
* RBAC and JWT authentication based on Spring Boot Security
* A complete and well-documented development workflow
* Pre-configured Docker middleware services that can be launched immediately
